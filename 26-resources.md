## Checkout Plugins

### Shopify

- Install from Shopify App Store
- Authenticate with ABA Merchant Portal credentials
- Supports ABA PAY, KHQR, Visa, Mastercard, UnionPay, JCB, WeChat Pay
- ABA PAY & KHQR settlements are typically instant
- Refunds processed through ABA PayWay Merchant Portal

### WooCommerce

- Install via WordPress Admin > Plugins > Add New Plugin > search "ABA PayWay"
- Configure with Merchant ID and API Key
- Supports ABA PAY, KHQR, Visa, Mastercard, UnionPay, JCB, Alipay, WeChat Pay

### PrestaShop

- Download plugin from `payway.com.kh/prestashop-v1.7.zip`
- Upload via PrestaShop Admin Panel > Modules
- Same payment methods as WooCommerce
